# Azure Fundamentals

## What is Azure
Microsoft Azure is a cloud computing platform that provides services like:
- Virtual machines
- Databases
- Web hosting
- Storage
- Networking

It allows companies to run applications without managing physical servers.

---

## What is Cloud Computing
Cloud computing means using remote servers over the internet to:
- Store data
- Run applications
- Manage infrastructure

Instead of using your own local machines.

---

## What is a Subscription
A subscription is a billing and management container in Azure.

- Every resource belongs to a subscription.
- It tracks usage and cost.

Example:
Subscription
└── Resource Group
    └── App Service

---

## What is a Resource Group
A resource group is a logical container that holds related Azure resources.

Example:
Resource Group: rg-azure-learning
├── App Service
├── Database
└── Storage

It helps in:
- Organizing resources
- Managing access
- Deleting resources together

---

## What is a Region
A region is a physical location of Azure data centers.

Examples:
- Central India
- East US
- West Europe

Resources are deployed into regions.

---

## What is Azure Resource Manager (ARM)
Azure Resource Manager is the deployment and management system for Azure.

It:
- Handles creation of resources
- Manages permissions
- Organizes resources into resource groups

All Azure resources are created through ARM.

---

## What is Azure DevOps
Azure DevOps is a set of tools used to manage software development.

It includes:
- Repos → source code storage
- Pipelines → build and deployment automation
- Boards → task and work tracking
- Artifacts → package management

It helps teams:
- Build code
- Test applications
- Deploy software automatically
